System Description

Smart Feeder Monitoring System

Version 1.0 (Prototype)

This system provides live monitoring of the automatic pet feeder using a local touchscreen interface. It includes a feeding schedule, food level sensor, and live camera feed capabilities.

Designed for: Tails of Freedom Animal Haven -- Silang, Cavite

Purpose: Internal use by shelter staff for efficient animal care management

User Guide

1
Tap "Dashboard" to view current feeder status and food levels
2
Use "Feed Now" button for manual feeding when necessary
3
The feeder operates automatically based on the programmed schedule
